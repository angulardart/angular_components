I18n

RTL

Use the flip attribute to automatically mirror an icon when the direction is RTL.

Icons listed in _flippedIcons in material_icon.dart are always flipped when the direction RTL. If an icon you're using should always be flipped for RTL, please add it to the list so others can benefit.

is automatically flipped when the direction is RTL, without using the flip attribute.

Not all icons should be flipped when the direction is RTL. In general, icons that deal with spatial relationships should be mirrored, while icons that represent temporal relationships, physical objects, or product logos should not be mirrored. Consult the material guidelines for detailed advice on when to flip icons.
